# SoaringTech 디스코드 봇

- 비상테크를 위한 디스코드 봇

# Installation

- Download `node.js` using `download_nodejs.sh`

```bash
download_nodejs.sh
```

- Source `node.js` environments

```
source enable_environments.sh
```

- Install npm packages

```
npm install
```
